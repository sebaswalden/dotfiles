# dotfiles

Personal dotfiles for macOS and Linux.

## Installation

```bash
git clone git@github.com:sebaswalden/dotfiles.git ~/dotfiles
cd ~/dotfiles
./setup.sh
```

## Post-installation

1. Restart terminal or run `zsh`
2. Set terminal font to "JetBrainsMono Nerd Font"

## Manual Setup

Remember to set up API keys for the following applications:

- [opencode](https://opencode.ai/docs)
- [claude-code](https://docs.anthropic.com/en/docs/claude-code)
